Game:
To run the game simply open a browser to the index.html file inside the www folder. 
You must have a PC that supports WebGL and JavaScript. A server is not required to run the
game.

Website:
The website is a wordpress CMS. It requires a server with php support and a mysql server to
host the database. Copy the contents of the website folder into your web server, upload the
database (wordpress.sql) to your mysql server, then change the wp-config.php file to match
your server setttings.
